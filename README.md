# Калькулятор
  Этот проект представляет собой простой калькулятор, реализованный на Java. Он позволяет выполнять основные арифметические операции (сложение, вычитание, умножение, деление) и вычислять факториал числа.
  Калькулятор работает через командную строку, запрашивая у пользователя необходимые данные для вычислений.

### Основные функции:
    Сложение (+) - складывает два введенных числа.
    Вычитание (-) - вычитает одно число из другого.
    Умножение (*) - умножает два введенных числа.
    Деление (/) - делит одно число на другое. Предусмотрена обработка деления на ноль.
    Факториал (!) - вычисляет факториал введенного целого числа.

### Установка и запуск:
    Скачайте IntelliJ IDEA
    Откройте проект в IntelliJ IDEA, Eclipse или другой Java-совместимой IDE
    Импортируйте все файлы .java
    Запустите программу в классе Main
